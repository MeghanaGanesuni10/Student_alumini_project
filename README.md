# Student-Alumni Management System

## Project Overview
This project is a Student-Alumni Management System developed to manage student and alumni information using a database and web interface.

## Features
- Add student and alumni records
- Update existing records
- Delete records
- View stored records
- Database management using CRUD operations

## Technologies Used
- HTML
- CSS
- JavaScript
- Database (MongoDB / SQL)
- Web Development

## Functionality
The system allows users to manage student and alumni data efficiently through a web interface with database connectivity.
